# musicNotes

A small Python project for learning how to turn sheet music into playable audio.

## Current goal

Build a first step that:
1. loads a real piece of music,
2. extracts one part,
3. writes a MIDI file, and
4. optionally renders audio if a soundfont is available.

## Planned phases

- Phase 1: render one part to MIDI/audio
- Phase 2: overlay multiple parts
- Phase 3: emphasize one line with audio tools
- Phase 4: add scale-degree and Roman numeral analysis

## Getting started

1. Create a virtual environment.
2. Install the Python dependencies.
3. Run the demo script.
